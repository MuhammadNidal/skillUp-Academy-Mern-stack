If Statement
An if statement checks a condition and runs code if true.

Syntax:
if (condition) {
  // code if true
}
Example:

javascript

Copy
let age = 20;
if (age >= 18) {
  console.log("You can vote.");
}
Output: "You can vote."

If...Else:
Runs alternative code if condition is false.

javascript

Copy
let age = 16;
if (age >= 18) {
  console.log("You can vote.");
} else {
  console.log("Too young to vote.");
}
Output: "Too young to vote."

If...Else If...Else:
Checks multiple conditions.

javascript

Copy
let marks = 75;
if (marks >= 90) {
  console.log("A+");
} else if (marks >= 80) {
  console.log("A");
} else if (marks >= 70) {
  console.log("B");
} else {
  console.log("C or below");
}
Output: "B"




Switch Statement

A switch statement is used to perform different actions based on different values.
Itâ€™s a cleaner alternative when you have many possible conditions for a single variable.

Syntax:

switch (expression) {
  case value1:
    // code if expression === value1
    break;
  case value2:
    // code if expression === value2
    break;
  default:
    // code if no case matches
}


Example:

let day = 3;

switch (day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  case 3:
    console.log("Wednesday");
    break;
  case 4:
    console.log("Thursday");
    break;
  case 5:
    console.log("Friday");
    break;
  default:
    console.log("Weekend");
}


Output:

Wednesday
