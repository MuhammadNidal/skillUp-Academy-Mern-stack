üß± 3. DOM Manipulation ‚Äì Add, Remove, Replace Elements

You can create, remove, or replace HTML elements using JavaScript.

‚úÖ Create an element
<p id="info">Old paragraph</p>

<script>
const newPara = document.createElement("p");
newPara.textContent = "This is new!";
document.body.appendChild(newPara); // adds it to the end of the body
</script>

‚úÖ Remove an element
<p id="remove">Remove me!</p>

<script>
const removeMe = document.getElementById("remove");
removeMe.remove(); // removes the element
</script>

‚úÖ Replace an element
<h1 id="main">Old Heading</h1>

<script>
const newHeading = document.createElement("h2");
newHeading.textContent = "I'm a new heading";

const oldHeading = document.getElementById("main");
document.body.replaceChild(newHeading, oldHeading);
</script>

üñ±Ô∏è 4. DOM Events ‚Äì Making Pages Interactive

Events help you respond to user actions (like clicks or typing).

‚úÖ Add Event Listener
<button id="btn">Click me</button>

<script>
document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});
</script>

‚úÖ Remove Event Listener
<button id="btn">Say Hi</button>

<script>
function sayHi() {
  alert("Hi!");
}

const btn = document.getElementById("btn");
btn.addEventListener("click", sayHi);

// Remove listener
btn.removeEventListener("click", sayHi);
</script>

üß† 7. InnerHTML vs textContent vs innerText
Property	Description	Example Output
textContent	Shows all text (including hidden)	"Hello World"
innerText	Shows only visible text	"Hello" (if ‚ÄúWorld‚Äù is hidden)
innerHTML	Includes HTML tags	"<b>Hello</b> World"

Example:

<div id="demo"><b>Hello</b> World</div>

<script>
const el = document.getElementById("demo");
console.log(el.textContent); // Hello World
console.log(el.innerText);   // Hello World (visible only)
console.log(el.innerHTML);   // <b>Hello</b> World
</script>
