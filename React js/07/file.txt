3Ô∏è‚É£ Fetch API (Built-in JavaScript)
üëâ What is Fetch?

Fetch is a built-in JavaScript function

Used to request data from APIs

Returns a Promise

‚úÖ Basic Fetch Example (Very Simple)
fetch("https://jsonplaceholder.typicode.com/posts")
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.log(error);
  });

üîç Line-by-Line Explanation
fetch("URL")


‚û° Sends request to the server

.then(response => response.json())


‚û° Converts server response into JSON

.then(data => console.log(data))


‚û° Uses the data

.catch(error => console.log(error))
‚û° Handles error if request fails

4Ô∏è‚É£ Fetch in React (Real Example)
import { useEffect, useState } from "react";

function Posts() {
  const [posts, setPosts] = useState([]);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/posts")
      .then(res => res.json())
      .then(data => setPosts(data))
      .catch(err => console.log(err));
  }, []);

  return (
    <div>
      <h2>Posts</h2>
      {posts.map(post => (
        <p key={post.id}>{post.title}</p>
      ))}
    </div>
  );
}

export default Posts;

5Ô∏è‚É£ Loading State (Very Important üî•)

üìå Problem:
Data takes time to load ‚Üí screen looks empty

üìå Solution:
Use a loading state

‚úÖ Example with Loading
function Posts() {
  const [posts, setPosts] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/posts")
      .then(res => res.json())
      .then(data => {
        setPosts(data);
        setLoading(false);
      });
  }, []);

  if (loading) {
    return <h3>Loading...</h3>;
  }

  return (
    <div>
      {posts.map(post => (
        <p key={post.id}>{post.title}</p>
      ))}
    </div>
  );
}

üß† Explain to Students:

loading = true ‚Üí show loader

Data loaded ‚Üí loading = false

6Ô∏è‚É£ Error Handling

üìå What if:

Internet is off?

Server is down?

Wrong URL?

üëâ App should not crash ‚ùå

‚úÖ Example with Error Handling
const [error, setError] = useState(null);

useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/posts")
    .then(res => {
      if (!res.ok) {
        throw new Error("Something went wrong");
      }
      return res.json();
    })
    .then(data => setPosts(data))
    .catch(err => setError(err.message));
}, []);

if (error) {
  return <h3>Error: {error}</h3>;
}

7Ô∏è‚É£ Axios (Alternative to Fetch)
üëâ What is Axios?

External library

Easier than fetch

Automatically converts JSON

Better error handling

üìå Install:

npm install axios

‚úÖ Axios Example
import axios from "axios";

useEffect(() => {
  axios
    .get("https://jsonplaceholder.typicode.com/posts")
    .then(res => {
      setPosts(res.data);
    })
    .catch(err => {
      console.log(err);
    });
}, []);

üîç Difference from Fetch
Fetch	Axios
Built-in	External library
Manual JSON	Auto JSON
More code	Cleaner code
8Ô∏è‚É£ Axios with Loading & Error (Best Practice)
function Posts() {
  const [posts, setPosts] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState("");

  useEffect(() => {
    axios
      .get("https://jsonplaceholder.typicode.com/posts")
      .then(res => {
        setPosts(res.data);
        setLoading(false);
      })
      .catch(() => {
        setError("Failed to fetch data");
        setLoading(false);
      });
  }, []);

  if (loading) return <h3>Loading...</h3>;
  if (error) return <h3>{error}</h3>;

  return (
    <div>
      {posts.map(post => (
        <p key={post.id}>{post.title}</p>
      ))}
    </div>
  );
}
