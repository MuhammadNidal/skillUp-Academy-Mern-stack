5ï¸âƒ£ Updating Objects in State
Example: User Profile
function Profile() {
  const [user, setUser] = useState({
    name: "Ali",
    age: 20,
  });

  const increaseAge = () => {
    setUser({
      ...user,
      age: user.age + 1,
    });
  };

  return (
    <div>
      <h2>{user.name}</h2>
      <p>Age: {user.age}</p>
      <button onClick={increaseAge}>Increase Age</button>
    </div>
  );
}

Explain:

...user â†’ copy old object

Only update required field
